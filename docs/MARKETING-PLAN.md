# STMIK Tazkia Marketing Plan 2026

## Executive Summary

STMIK Tazkia is a new IT college in Bogor targeting 3,000 leads per admission cycle with 10% conversion (300 registrations). This plan outlines a comprehensive marketing strategy leveraging the institution's unique positioning: **project-based learning with real industry projects** and **halal tech ecosystem focus**.

**Budget Assumption:** Cost-effective approach suitable for new institution
**Timeline:** 12-month rolling plan with quarterly reviews

---

## Part 1: Situation Analysis

### 1.1 Market Position

| Factor | Status |
|--------|--------|
| Institution Type | Private IT College (STMIK) |
| Location | Bogor, West Java |
| Programs | S1 Teknik Informatika, S1 Sistem Informasi |
| Accreditation | Active (SK 078/SK/LAM-INFOKOM/2024) |
| Parent Organization | Yayasan Tazkia Cendikia (established brand in Islamic finance education) |

### 1.2 Unique Value Propositions (UVP)

1. **Project-Based Learning** - 4 courses integrated into 1 real industry project per semester
2. **Halal Tech Ecosystem** - Niche positioning in growing halal industry
3. **Industry Internship** - 12 SKS mandatory internship (Teknik Informatika)
4. **Tazkia Brand** - Leverages parent brand's reputation in Islamic education
5. **Flexible Schedule** - Accommodates working students
6. **Affordable** - Competitive pricing with scholarship options

### 1.3 SWOT Analysis

| Strengths | Weaknesses |
|-----------|------------|
| Tazkia brand recognition | New institution (no alumni track record) |
| Unique project-based curriculum | Limited brand awareness as IT college |
| Industry partnerships | Small faculty size initially |
| Islamic values integration | No campus facilities showcase yet |
| Competitive pricing | Limited online presence |

| Opportunities | Threats |
|---------------|---------|
| Growing IT job market | Established competitors (IPB, UI, Pakuan) |
| Halal industry growth ($7T global) | Free online courses (Coursera, etc.) |
| Remote work trend | Economic uncertainty affecting enrollment |
| Gen Z preference for practical skills | Government scholarship programs at state universities |
| Underserved Bogor market | Perception: "Islamic = not tech-focused" |

### 1.4 Competitive Landscape (Bogor Region)

| Competitor | Type | Strength | Weakness |
|------------|------|----------|----------|
| IPB University | State | Brand, research | Not IT-focused |
| Universitas Pakuan | Private | Established | Traditional curriculum |
| STMIK Triguna Dharma | Private | IT-focused | Limited industry connection |
| Telkom University (nearby) | Private | Tech reputation | Higher cost, Bandung location |

**Competitive Advantage:** No other institution in Bogor offers project-based IT curriculum with halal tech focus.

## Part 2: Target Audience

### 2.1 Primary Segments

#### Segment A: Fresh High School Graduates (60% of target)
- **Demographics:** 17-19 years old, SMA/SMK/MA graduates
- **Location:** Bogor, Depok, Jakarta Selatan, Sukabumi, Cianjur
- **Psychographics:**
  - Interested in technology/gaming/social media
  - Value practical skills over theory
  - Budget-conscious families
  - Some prefer Islamic environment
- **Decision Influencers:** Parents (70%), peers (20%), teachers (10%)
- **Search Behavior:** Instagram, TikTok, Google, YouTube
- **Pain Points:**
  - Unsure about career path
  - Worried about job prospects after graduation
  - Can't afford expensive private universities
  - Want hands-on learning, not just theory

#### Segment B: Working Professionals (30% of target)
- **Demographics:** 20-35 years old, employed in non-IT roles
- **Location:** Greater Bogor area, willing to commute
- **Psychographics:**
  - Want career change to IT
  - Need flexible schedule (evening/weekend classes)
  - Value efficiency and practical outcomes
  - Self-funded or employer-sponsored
- **Search Behavior:** Google, LinkedIn, WhatsApp groups
- **Pain Points:**
  - Limited time for full-time study
  - Need recognized degree (not just bootcamp certificate)
  - Want ROI on education investment
  - Concerned about age/fitting in

#### Segment C: Parents of Prospective Students (Influencer)
- **Demographics:** 40-55 years old, middle-class families
- **Psychographics:**
  - Want best future for children
  - Value job security and clear career path
  - Appreciate Islamic values in education
  - Cost-sensitive but willing to invest
- **Search Behavior:** Google, WhatsApp, word-of-mouth
- **Pain Points:**
  - Worried about children's job prospects
  - Confused by many university options
  - Want value for money
  - Concerned about campus environment/safety

### 6.3 Technical SEO Improvements

#### Priority 1: Critical Fixes (Week 1-2)

| Issue | Current | Action |
|-------|---------|--------|

| `twitter:image` missing | No Twitter card image | Add same image |
| Canonical URLs | Not implemented | Add per-page canonical links |
| hreflang | Only homepage | Make dynamic per-page |
| robots.txt | Blocking all | Update when ready to launch |
| noindex flag | Enabled | Disable when ready |

**Implementation:**

```astro
<!-- BaseLayout.astro additions -->
<link rel="canonical" href={canonicalURL} />
<meta property="og:image" content={ogImage} />
<meta property="og:url" content={canonicalURL} />
<meta name="twitter:image" content={ogImage} />
<link rel="alternate" hreflang="id" href={idURL} />
<link rel="alternate" hreflang="en" href={enURL} />
<link rel="alternate" hreflang="x-default" href={idURL} />
```

### 6.5 Conversion Rate Optimization

#### Current Issues

1. **Weak CTAs** - Generic "Daftar Sekarang" without urgency
2. **No lead magnets** - No reason to give contact info early
3. **Single conversion path** - Only form, no alternatives
4. **No social proof on key pages** - Missing testimonials near CTAs

#### Recommended Improvements

**A. Enhance CTA Design**

| Current | Improved |
|---------|----------|
| "Daftar Sekarang" | "Daftar Sekarang - Gelombang 1 Diskon 20%" |
| "Pelajari Lebih Lanjut" | "Lihat Kurikulum & Biaya" |
| "Konsultasi Gratis" | "Chat WhatsApp Sekarang (Balas < 5 menit)" |

**B. Add Lead Magnets**

| Lead Magnet | Page | Capture |
|-------------|------|---------|
| "Panduan Karir IT 2026" (PDF) | Homepage, Programs | Email + WhatsApp |
| "Simulasi Biaya Kuliah" | Admissions | WhatsApp |
| "Checklist Persiapan Kuliah" | Admissions | Email |

**C. Multiple Conversion Points**

```
Homepage:
├── Hero: Primary CTA (Daftar)
├── Mid-page: Secondary CTA (Konsultasi WhatsApp)
├── Program section: Tertiary CTA (Lihat Detail)
└── Footer: Sticky mobile CTA bar

Program Pages:
├── Hero: CTA (Daftar Program Ini)
├── After curriculum: CTA (Download Kurikulum)
├── After careers: CTA (Konsultasi Karir)
└── Bottom: Final CTA (Daftar Sekarang)
```

**D. Trust Elements**

Add near CTAs:
- Accreditation badge
- Student count / satisfaction stat
- "Sudah 500+ mahasiswa terdaftar" (when applicable)
