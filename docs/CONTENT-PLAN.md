## Content Guidelines

### SEO Best Practices
- Include target keyword in title and first paragraph
- Use H2/H3 headings with related keywords
- Add internal links to program pages and contact
- Include call-to-action (CTA) for registration
- Optimal length: 1000-2000 words for articles

### Image Requirements
- Featured image: 1200x630px (OG image size)
- In-article images: 800px width max
- Alt text with keywords
- Compress before upload

### Publishing Workflow
1. **Draft** - Writer creates initial content
2. **Review** - Editor reviews for accuracy and SEO
3. **Publish** - Upload to website with proper metadata
4. **Promote** - Share on social media channels
